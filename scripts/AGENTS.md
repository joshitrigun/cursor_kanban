This folder contains start and stop scripts for Windows, macOS, and Linux.

- `start-windows.ps1` and `stop-windows.ps1` manage the Docker container from PowerShell.
- `start-macos.sh` and `stop-macos.sh` manage the Docker container from macOS.
- `start-linux.sh` and `stop-linux.sh` manage the Docker container from Linux.

All scripts target the same image and container name so the usage stays consistent across platforms.
